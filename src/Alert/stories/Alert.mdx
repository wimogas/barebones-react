# Alert

The alert component is used to display a message to the user. It can be used to display a success message, an error message, in banner or toast format.

<Canvas of={AlertStories.Default} />

## Usage

```jsx
import { Alert } from 'react-barebones-ts';

<Alert>Alert message</Alert>

```

## Types

### banner

Usage: The alert of type banner is used to display a message to the user which has higher priority than the other alert messages.
```jsx
<Alert type="banner">Banner message</Alert>

```

<Canvas of={AlertStories.Banner} />

### toast

Usage: The alert of type toast is used to display a message to the user which has lower priority than the other alert messages.

```jsx
<Alert type="toast">Toast message</Alert>

```

<Canvas of={AlertStories.Toast} />

## Custom Icons

Usage: The alert can be customized with custom icons using the icon and iconColor prop.

```jsx
<Alert icon={<UserIcon />} iconColor='#CCCCCC'/>

```

## Properties

### classes

Your custom classes for the Alert.

```jsx
<Alert classes={"bb-bg-primary-500 bb-white"}>Alert alert!</Alert>
```

### style

Your custom React style object.

```jsx
<Alert style={{"backgroundColor": "#000"}}>Alert alert!</Alert>
```
