## CSS Framework

Documentation coming soon!

Some basics:

```
// Colors
.bb-color-primary-{100-900}

// Backgrounds
.bb-background-{01-15}

// Borders
.bb-border-primary-{01-06}

// Layout
.bb-block
.bb-row
.bb-column
.bb-gap-{100-300}
.bb-justify-{start|end|center|between|around}
.bb-align-{start|end|center|baseline|stretch}
.bb-w-{25|33|50|75|100}
.bb-hide

// Grid columns
.bb-col-{1-12}

// Padding
.bb-p-{50-950}
Also valid for specific directions:
.bb-p{t|b|l|r}-{50-950}

// Margin
.bb-m-{50-950}
Also valid for specific directions:
.bb-m{t|b|l|r}-{50-950}

// Typography
.bb-font-size-{100-900}
.bb-line-height-{100-900}
.bb-font-{bold|semibold|regular}
.bb-h{1-6}-{bold|semibold|regular}
.bb-body-{large|medium|small}-{bold|semibold|regular}
```
