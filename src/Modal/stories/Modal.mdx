# Modal

The modal component is used to display content in a layer above the app.

<Canvas of={ModalStories.Launch} />

## Usage

```jsx
import { Modal } from 'react-barebones-ts';

<Modal title="Modal title" close={handleSetShowModal}>
  ...
</Modal>
```

## Props

### children

The content of the modal.

```jsx

<Modal>
  Modal content
</Modal>

```
