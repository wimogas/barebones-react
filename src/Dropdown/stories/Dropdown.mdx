# Dropdown

A dropdown component is used to select an item from a list of options.

<Canvas of={DropdownStories.Right} />

## Usage

```jsx
import { Dropdown } from 'react-barebones-ts';

const dropdownMenuItems = [
    <Button key="1" action={() => {}}>Item 1</Button>,
    <Button key="2" action={() => {}}>Item 2</Button>,
];

<Dropdown
    buttonChildren="Dropdown"
    items={dropdownMenuItems}
/>

````

## Props

### buttonChildren

The Children to display inside the Button. Can be a string or a React Node.

```jsx
<Dropdown buttonChildren={"Dropdown"}/>
```

### buttonStyle

The Style of the Button.

```jsx
<Dropdown buttonStyle={{"backgroundColor":"black"}}/>
```

### buttonClasses

The Classes of the Button.

```jsx
<Dropdown buttonClasses={"bb-bg-primary-500"}/>
```

### menuClasses

The Classes of the Dropdown Menu.

```jsx
<Dropdown menuClasses={"bb-bg-primary-500"}/>
```

### menuStyle

The Style of the Dropdown Menu.

```jsx
<Dropdown menuStyle={{"backgroundColor" : "black"}}/>
```

### items

Array of React Nodes


```jsx
const dropdownMenuItems = [
    <Button key="1" action={() => {}}>Item 1</Button>,
    <Button key="2" action={() => {}}>Item 2</Button>,
];

<Dropdown items={dropdownMenuItems} />
```

### direction

The direction prop determines the direction that the dropdown menu is displayed. It can be one of the following values:

types: `right` | `left`

```jsx
<Dropdown direction="left" />
```
