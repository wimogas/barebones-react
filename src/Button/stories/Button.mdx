# Button component

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

<Canvas of={ButtonStories.Default} />

## Usage

```jsx
import { Button } from 'react-barebones-ts';

<Button>Button Default</Button>

```

## Properties

### classes

Your custom classes for the button.

```jsx
<Button classes={"bb-bg-primary-500 bb-white"}>Click me</Button>
```

### style

Your custom React style object.

```jsx
<Button style={{"backgroundColor": "#000"}}>Click me</Button>
```

### icon

The icon of the button. Pass an SVG or \<i\> as a React Component.

```jsx
<Button icon="<CheckIcon/>">Click me</Button>
```

### disabled

The disabled state of the button.

```jsx
<Button disabled>Can't click me</Button>
```

### action

The click handler of the button.

```jsx
<Button action={() => alert('Clicked!')}>Click me</Button>
```

### children

The content of the button.

```jsx
<Button>{children}</Button>
```
