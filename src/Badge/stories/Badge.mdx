# Badge

Badges are used to inform the user of an item's status or value.

<Canvas of={BadgeStories.Default} />

## Usage

```jsx
import { Badge } from 'react-barebones-ts';

<Badge>Badge</Badge>
```

## Properties

### classes

Your custom classes for the Badge.

```jsx
<Badge classes={"bb-bg-primary-500 bb-white"}>Custom badge</Badge>
```

### style

Your custom React style object.

```jsx
<Badge style={{"backgroundColor": "#000"}}>Custom badge</Badge>
```
