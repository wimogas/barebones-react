# Text

The Text component is used to render text in the UI.

## Usage

```jsx
import { Text } from 'react-barebones-ts';

<Text text="Text to display"/>

```

## Props


### text

The text to render.

```jsx
<Text text="Text to display"/>

```

### type

The type of text to render.

```jsx
<Text type="span">Text</Text>

```

### size

The size of the text.

types: 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900

```jsx
<Text size={500}>Text</Text>

```

### bold

Whether the text should be bold.

```jsx
<Text bold>Text</Text>

```

### color

The color of the text.

```jsx
<Text color="primary">Text</Text>

```
