# Text

The Text component is used to render text in the UI.

## Usage

```jsx
import { Text } from 'react-barebones-ts';

<Text text="Text to display"/>

```

## Props

### classes

Your custom classes for the Text.

```jsx
<Text classes={"bb-primary-200"}>Hello world</Text>
```

### style

Your custom React style object.

```jsx
<Text style={{"color": "#000"}}>Hello world</Text>
```

### text

The text to render.

```jsx
<Text text="Text to display"/>

```

### type

The type of text to render.

```jsx
<Text type="span">Text</Text>

```
